# Why two repos?

Once upon a time, this project was closed source.
Reason: a studio whose name starts with the letter 'S' and rhymes with "Parfish Pudios" [openly and seriously floated the idea of stealing my features](https://discord.com/channels/1382334723333820568/1382334724131000395). Yes, really.
That situation cooled off, months passed, nothing happened, and I had more productive things to do than run a Cold War over hamsters.

Later, translation support via Crowdin required the project to be open source to use Crowdin's free plan.
So the code returned to the light, but the assets stayed protected.
